# UI界面使用说明

## 安装依赖

```bash
pip install PyQt6>=6.0.0
```

或者使用项目的requirements.txt：

```bash
pip install -r ../requirements.txt
```

## 运行UI

### 方法1：直接运行主窗口

```bash
python -m UI_interface.main_window
```

### 方法2：使用启动脚本

```bash
python -m UI_interface.run_ui
```

## 功能说明

### 界面布局

- **左侧面板**：
  - 顶部：程序名称"三角洲自动剪辑软件"
  - 底部：三个按钮（设置、打断、启动）

- **右侧面板**：
  - 可滚动的Banner列表区域
  - 每个Banner显示视频信息和处理状态

### Banner功能

1. **空白Banner**：
   - 显示灰色占位图
   - 标题："拖入视频或选择视频"
   - 包含"选择文件"按钮

2. **视频Banner**：
   - 左侧：视频封面图（当前为占位图）
   - 右侧：
     - 第一行：视频名称
     - 第二行：视频时长和分辨率
     - 第三行：视频路径
   - 最右侧：删除按钮（×）
   - 最左侧：序号（从1开始）

3. **文件添加方式**：
   - 点击"选择文件"按钮，支持多选
   - 直接拖放视频文件到窗口，支持多选

4. **处理状态**：
   - **未开始**：无状态图标
   - **等待中**：右下角显示等待图标（三个点）
   - **处理中**：
     - 背景从左到右填充绿色进度条
     - 右下角显示播放图标
   - **已完成**：
     - 进度条填满
     - 第三行显示"已保存到：xxx（路径）"

### 操作说明

1. **添加视频**：
   - 点击空白Banner的"选择文件"按钮
   - 或直接拖放视频文件到窗口

2. **删除视频**：
   - 点击Banner右侧的"×"按钮

3. **开始处理**：
   - 点击左下角的绿色"启动"按钮
   - 系统会按从上到下的顺序处理所有视频
   - 每个视频处理时间为5秒（占位）

4. **打断处理**：
   - 点击"打断"按钮可以停止当前处理

5. **设置**：
   - 点击"设置"按钮（功能暂未实现）

## 注意事项

- 当前视频封面图使用占位图，实际应该从视频提取首帧
- 视频信息（时长、分辨率）当前为模拟数据，实际应该从视频文件读取
- 处理逻辑使用5秒占位，实际应该接入真实的视频处理逻辑
